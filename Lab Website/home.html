Christofk Lab Internal Site

Mass Spec RT Database Allows lab members to access and download lists of compounds from our internal database with their m/z and retention time values to identify peaks in MS data.
Mass calculator Allows lab members to search for compounds in Pubchem and calculate the m/z for the M0 forms and various (M + n) carbon isotopomers.
Metabolomics Queue Allows lab members and collaborators to add their samples to the metabolomics queue.
XCalibur Run Creator A tool that allows for the rapid creation of sample run files for the Thermo control software XCalibur.
Western Blot Assistant A tool that allows for looking up of protein sizes in kD and plotting them visually on a western blot.
Chromatogram Viewer A tool that allows select chromotograms to isolated and plotted from mzXML files.
Christographa A tool that allows for graphing and analyzing metabolomics data.
